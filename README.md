# README

## OE-NIK Szakdolgozat / Diplomamunka LaTeX template

### Preliminary pages

A szakdolgozat/diplomamunka néhány kötelező oldallal indul, mint a borító, feladatlap, hallgatói nyilatkozat, konzultációs napló és (szükség esetén) a titoktartási kérelem.

Ezeknek a kinézetére szigorú előírások vonatkoznak. A pontos lemásolásuk (különösen a borító esetén) nehézkes, ráadásul minden egyes változtatás során újra elvégzendő feladat lenne. Emiatt azt a megoldást választottam, hogy mivel az aktuális hivatalos mellékletet minden bizonnyal közzéteszik valamilyen Word számára feldolgozható formátumban (ezen sorok írásakor .docx), ezért az eredeti mintát kitöltve, majd PDF formátumban elmentve könnyedén importálható.

### variables

A `variable.tex` fájlban található értékek kitöltése a végső PDF fájl adatait adja meg, ne felejtsd el kitölteni.

### abbreviations

Ott lehet megadni rövidítéseket, amiket automatikusan jegyzékbe gyűjt.
